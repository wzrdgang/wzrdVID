DOWNLOAD DECK

Desktop app is the full version.

Run the full local renderer on your Mac. Multi-source timelines, HEIC/photo motion loops, worky audio texture, chunky ANSI, real PUBLIC ACCESS VHS rendering, endings, size targets, and phone-sendable exports.

Download WZRD.VID for macOS

Use the latest ZIP from GitHub Releases. Intel Macs can run from source for now.

  1. Download WZRD.VID for macOS
  2. Unzip
  3. Open WZRD.VID.app
  4. Right-click Open if macOS warns
  5. Install ffmpeg if prompted: brew install ffmpeg

Updates work the same way: download the latest ZIP and replace the old app manually. WZRD.VID can notify you, but it does not auto-install updates yet.
