# Examples

This folder is for public-safe screenshots, GIFs, demo presets, and small generated examples.

Suggested layout:

```text
examples/
  screenshots/
  demos/
  presets/
```

Do not commit copyrighted music, music videos, movie clips, personal footage, or sample media you do not have rights to redistribute. Prefer generated clips, still images you own, or short outputs made from clearly licensed source media.

If a demo needs external media, document where users can place their own files instead of committing those files to the repository.
